# Azure Learning

Dieses Repository enthält eine Sammlung von Azure-Lernpfaden, Dokumentationen und praktischen Bereitstellungsbeispielen. Es dient der strukturierten Vermittlung von Azure-Konzepten, Architekturprinzipien und Umsetzungsbeispielen.

## Repository-Übersicht

Das Repository enthält:

- [docs/](docs/) mit Lernpfaden und Dokumentationen.
- [deployments/](deployments/) mit Bereitstellungsdateien und Skripten.

## Verfügbare Lernpfade

- [docs/01-dmz-private-workload-architecture/README.md](docs/01-dmz-private-workload-architecture/README.md) – Lernpfad zur DMZ- und Private-Workload-Architektur mit Fokus auf Netzwerksegmentierung, NSGs, virtuellen Maschinen, Hardening und Wartungszugriffen.

## Aufbau eines Lernpfads

Ein Lernpfad besteht aus:

- einer Einstiegseite im Ordner [docs/](docs/),
- mehreren nummerierten Kapiteln zu den jeweiligen Themen,
- und optionalen Deployment-Dateien im Ordner [deployments/](deployments/), die die beschriebenen Schritte nachvollziehbar abbilden.

## Nutzung

1. Ein Lernpfad wird über den entsprechenden Ordner in [docs/](docs/) ausgewählt.
2. Die Kapitel werden in der beschriebenen Reihenfolge bearbeitet.
3. Die zugehörigen Dateien in [deployments/](deployments/) werden für die praktische Umsetzung verwendet.

## Ziel

Das Repository stellt eine nachvollziehbare Verbindung zwischen Azure-Theorie, Architekturentscheidung und konkreter Bereitstellung bereit.
